Portfolio website

September 19, 2019

Hello world!

This is a post about this very site. This is a part of the training in the CoderAcademy sowing some seeds for a fully fledged portfolio. This is also the first site I am working on for real. The start was rough - even though HTML5 and CSS3 are easy to learn, making something which was not ugly was really hard for me. After a day of flexing divs around I decided to use a framework. I considered Bootstrap and Material Design, the latter looked too ephemeral and scary.

Next day I got something which was not ugly and I was really happy with the amount of code it required. The page did not put you into aesthetic coma any more, but did not stand out either. I decided to change theme. Being very bad at Graphic Design, I searched for “colour schemes for web-sites” and got a bunch of results. This part was easy. On the contrary, changing theme in Bootstrap was not easy, tweaking the styles with my own kept exploding bringing the site to the previous levels of ugliness.

I had to make it right - to add extra colours to the theme and to re-compile Bootstrap CSS file. Again I didn’t work with Node.js and corresponding packet manager before, so it took a lot of time to figure out, but I could generate the CSS at the end. It was not minimized, but that was just good enough. This took me another day.

When I was too angry about documentation or weather to learn new stuff I was working on demos of the two humble apps I had written before. I needed to record the demos and to convert them into gifs making them pleasantly looking. This was not as simple as I expected as I needed to make the demos short and crisp. I spent another day writing blog posts and adding content, overriding styles and recompiling Bootstrap. There was an enormous number of small things to add and to fix. HTML code duplication started to annoy me a lot - I kept forgetting to update things in all places they were used. I also didn’t enjoy typing tags, so I used Hugo to generate web pages from templates.

Now it’s time to polish the site a little bit more and publish it.
